# Отчет о релизе v1.3.0 - RenameMedia

В версии 1.3.0 мы реализовали ключевые функции для настройки именования и организации медиаархива.

## Основные изменения

### 1. Пользовательские шаблоны имен
Добавлен модуль `template_parser.py`, позволяющий пользователям задавать собственные форматы через теги:
- `{type}` — Photo или Video
- `{YYYY}`, `{MM}`, `{DD}` — год, месяц, день
- `{HH}`, `{mm}`, `{ss}` — часы, минуты, секунды

### 2. Организация по папкам
Добавлен модуль `folder_organizer.py`, который автоматически распределяет файлы в структуру:
`Базовая_папка/ГГГГ/ММ/Файл.jpg`

### 3. Интеграция в GUI
Интерфейс `rename_media_gui.py` был расширен:
- Добавлено **поле ввода шаблона** с подсказками по тегам.
- Добавлен **чекбокс для группировки по папкам**.
- Обновлена логика обработки: во время работы настройки блокируются, статистика учитывает перемещение файлов.

## Документация и проектные файлы
- **CHANGELOG.md**: Задокументированы изменения 1.3.0 и 1.2.0.
- **README.md**: Обновлены возможности и примеры использования.
- **pyproject.toml**: Версия поднята до 1.3.0, добавлены новые модули.
