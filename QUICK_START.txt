🎯 ЧТО ЭТО?
───────────
Программа автоматически переименовывает фото и видео по дате съёмки.

Было:           IMG_1234.jpg, VID_5678.mp4
Стало:          Photo-2023-08-15_142203.jpg, Video-2023-08-15_153045.mp4


⚡ САМЫЙ БЫСТРЫЙ СПОСОБ (для новичков):
───────────────────────────────────────

1. Установите Python 3.8+ с сайта https://python.org
   ⚠️ При установке поставьте галочку "Add Python to PATH"!

2. Откройте командную строку в папке с программой:
   - Зажмите Shift + Правая кнопка мыши на папке
   - Выберите "Открыть окно PowerShell здесь" или "Открыть командную строку"

3. Установите библиотеки:
   pip install -r requirements.txt

4. Запустите программу:
   python rename_media_gui.py

5. В открывшемся окне:
   [📁 Выбрать папку] ← Нажмите и выберите папку с фото/видео
   [☑] Тестовый режим ← Оставьте галочку для первого раза!
   [▶️ Запустить] ← Нажмите чтобы начать

6. Посмотрите результаты в окне, если всё ок:
   - Снимите галочку "Тестовый режим"
   - Нажмите "Запустить" снова


💻 ДЛЯ ПРОДВИНУТЫХ (командная строка):
──────────────────────────────────────

# Тестовый запуск (просмотр без изменений):
python rename_media_cli.py "C:\Мои фото" --test

# Реальное переименование:
python rename_media_cli.py "C:\Мои фото"

# Справка:
python rename_media_cli.py --help
