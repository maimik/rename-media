📌 ЧТО ДЕЛАЕТ ЭТА ПРОГРАММА?

Программа автоматически переименовывает ваши фото и видео файлы по дате,
когда они были сняты. Например:

  Было:           IMG_1234.jpg
  Стало:          Photo-2023-08-15_142203.jpg

  Было:           VID_20230815.mp4
  Стало:          Video-2023-08-15_142203.mp4


═══════════════════════════════════════════════════════════════════════════
📋 ШАГ 1: УСТАНОВКА PYTHON (если ещё не установлен)
═══════════════════════════════════════════════════════════════════════════

1. Перейдите на сайт: https://www.python.org/downloads/
2. Скачайте последнюю версию Python (3.8 или выше)
3. Запустите установщик
4. ⚠️ ВАЖНО: Поставьте галочку "Add Python to PATH" при установке!
5. Нажмите "Install Now"
6. Дождитесь завершения установки


═══════════════════════════════════════════════════════════════════════════
📦 ШАГ 2: УСТАНОВКА НЕОБХОДИМЫХ БИБЛИОТЕК
═══════════════════════════════════════════════════════════════════════════

1. Откройте командную строку (CMD):
   - Нажмите Win+R
   - Введите: cmd
   - Нажмите Enter

2. Перейдите в папку с программой:
   cd "путь\к\папке\Claude_Rename_media"

3. Установите библиотеки командой:
   pip install -r requirements.txt

4. Дождитесь завершения установки


═══════════════════════════════════════════════════════════════════════════
🎨 ШАГ 3: ЗАПУСК ПРОГРАММЫ (GUI - с окошками)
═══════════════════════════════════════════════════════════════════════════

ЭТО САМЫЙ ПРОСТОЙ СПОСОБ ДЛЯ НАЧИНАЮЩИХ!

1. Откройте командную строку в папке с программой (см. ШАГ 2)

2. Запустите команду:
   python rename_media_gui.py

3. Откроется окно программы:

   [📁 Выбрать папку]

   ↓ Нажмите эту кнопку и выберите папку с вашими фото/видео

   [☑] Тестовый режим (предпросмотр без изменений)

   ↓ Сначала оставьте эту галочку - программа покажет что будет изменено,
     но не будет менять файлы по-настоящему

   [▶️ Запустить переименование]

   ↓ Нажмите эту кнопку чтобы начать

4. После первого запуска в тестовом режиме:
   - Посмотрите что будет изменено
   - Если всё устраивает - снимите галочку "Тестовый режим"
   - Запустите ещё раз для реального переименования


═══════════════════════════════════════════════════════════════════════════
💻 ШАГ 3 (АЛЬТЕРНАТИВА): ЗАПУСК ИЗ КОМАНДНОЙ СТРОКИ
═══════════════════════════════════════════════════════════════════════════

ДЛЯ ТЕХ, КТО УМЕЕТ РАБОТАТЬ С КОМАНДНОЙ СТРОКОЙ:

1. Тестовый запуск (без изменений):
   python rename_media_cli.py "C:\Мои фото" --test

2. Реальное переименование:
   python rename_media_cli.py "C:\Мои фото"

3. Справка:
   python rename_media_cli.py --help

═══════════════════════════════════════════════════════════════════════════
⚠️ ВАЖНЫЕ ПРЕДУПРЕЖДЕНИЯ!
═══════════════════════════════════════════════════════════════════════════

1. ⚠️ СДЕЛАЙТЕ РЕЗЕРВНУЮ КОПИЮ ВАШИХ ФАЙЛОВ ПЕРЕД ИСПОЛЬЗОВАНИЕМ!
   Скопируйте папку с фото/видео в безопасное место.

2. ⚠️ Сначала ВСЕГДА запускайте в тестовом режиме!
   Это позволит увидеть что будет изменено без реальных изменений.

3. ⚠️ Программа переименовывает файлы в той же папке где они находятся.
   Она НЕ перемещает файлы в другие места.

4. ⚠️ Если у файла нет метаданных о дате съёмки:
   Будет использована дата создания файла из Windows.


═══════════════════════════════════════════════════════════════════════════
🔧 ДОПОЛНИТЕЛЬНАЯ НАСТРОЙКА (для видео)
═══════════════════════════════════════════════════════════════════════════

Для лучшей работы с ВИДЕО рекомендуется установить ffmpeg:

1. Перейдите на: https://ffmpeg.org/download.html
2. Скачайте версию для Windows
3. Распакуйте архив
4. Добавьте путь к ffmpeg.exe в переменную PATH:
   - Найдите "Переменные среды" в настройках Windows
   - Добавьте путь к папке с ffmpeg.exe в PATH
   - Перезапустите командную строку

⚠️ Без ffmpeg для видео будет использоваться дата файла из Windows,
   а не реальная дата съёмки из метаданных видео.

❓ ЧАСТО ЗАДАВАЕМЫЕ ВОПРОСЫ
═══════════════════════════════════════════════════════════════════════════

Q: Программа не запускается, пишет "Python не найден"
A: Переустановите Python и обязательно поставьте галочку "Add Python to PATH"

Q: Как отменить переименование?
A: Если вы не делали резервную копию - отменить нельзя.
   Поэтому ВСЕГДА делайте резервную копию!

Q: Программа пропускает некоторые файлы
A: Возможно у этих файлов нет метаданных EXIF или они повреждены.
   Проверьте лог в окне программы.

Q: Что делать если два файла имеют одинаковую дату?
A: Программа автоматически добавит номер: Photo-2023-08-15_142203_1.jpg

Q: Можно ли запустить на Mac или Linux?
A: Да, программа работает на всех системах где есть Python.

Q: Как создать EXE файл чтобы не устанавливать Python?
A: См. инструкцию в файле CLAUDE.MD (раздел про PyInstaller)
